# FE-Challenge

## Prerequisites

Node v18 or higher.

## Run application

Please use the command below to run the application

```bash

git clone https://github.com/pj-dev9106/fe-challenge

cd fe-challenge

npm install

npm run dev

```

After run the application, it should be run on the port 3000: http://localhost:3000
